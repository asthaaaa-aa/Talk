
# Talk - AI mental health assistant with sentiment analysis

> A small assistant-style app with a React + Vite frontend, a Node/Express backend, and a local ML service for finding the sentiment of the chats.

---

## Repository layout

- `frontend/` - Vite + React SPA (UI)
- `backend/` - Node + Express API server
- `ml-model/` - Python service (sentiment analysis logic)

## Features

- Chat UI using React
- Backend chat routes and thread model (MongoDB)
- Utilities for calling external LLMs / embeddings
